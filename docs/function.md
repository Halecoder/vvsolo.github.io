### 全局

> 整理：一次性规范文本格式，是所有细分功能的集合处理方式<br />排版：整理后的文本，一次性排版

- 00.整理 → 全局

  > 全局整理集合大部分常规转换，一般只需**执行一次**即可

  - HTML 字符实体转换
  - Unicode转换
  - 排版初始化，去空格空行
  - 去乱码、广告、不可读字符
  - 半角字母数字
  - 修正章节标题，包括非常规的章节标题
  - 修正错误标题
  - 全角标点符号
  - 去除汉字间的空格
  - 修正分隔符号
  - 修正引号为西式引号 `“”‘’`
  - 修正英文
  - 其他自定义修正

- 01.整理 → 增强

  > 增强整理处理段落和引用间的错误换行<br />若检测到双单引号非成对时，将中断并在右侧大纲视图中提示错误

  - 修正章节标题，包括非常规的章节标题
  - 修正错误标题
  - 去除汉字间的空格
  - 修正分隔符号
  - 半角字母数字
  - 修正引号为西式引号 `“”‘’`
  - 修正段落间的错误换行
  - 修正引号内的错误换行

- 02.整理 → 普通

  > 基本整理，以去除段落前后空格为主

  - Unicode转换
  - 排版初始化，去空格空行
  - 修正章节标题，不包括非常规的章节标题
  - 修正错误标题
  - 去除汉字间的空格
  - 修正分隔符号
  - 其他自定义修正

- 11.整理 → 特殊

  > 整理无任何排版规则的文本，一般指并强制分段的网页文本

  - HTML 字符实体转换
  - Unicode转换
  - 转换变体字母
  - 转换变体序号
  - 排版初始化，去空格空行
  - 去除汉字间的空格
  - 半角字母数字
  - 去乱码、广告、不可读
  - 保护书名、作者、无结尾标点的歌词不换行
  - 修正章节外加括号
  - 保护章节最后是句号的
  - 保护章节标题
  - 修正引号
  - 修正错误换行
  - 其他自定义修正
  - 去除汉字间的空格
  - 修正分隔符号
  - 再次修正章节标题

- 12.整理 → 分段反向

  - 反向整理以规则分段的文本，比如 35 字分段排版

- 21.排版 → 阅读样式
  - 段落前两个全角空格
  - 段落间不间隔行
  - 章节标题间隔两行

- 22.排版 → 论坛样式
  - 段落前两个全角空格
  - 段落间间隔一行

- 23.排版 → 网页样式
  - 段落前顶格无空格
  - 段落间间隔一行
  - 中文间无空格
  - 英文和数字与中文间美化空格

- 24.排版 → 分段样式
  - 段落按 35 字（可自定义）分隔换行
  - 段落首行前两个全角空格
  - 分隔部分不换行
  - 段落间间隔一行
  - 章节标题居中
  - 分隔符居中
  - 结尾标注性文本居中

### 转换

- 01.标题 → 大写转数字

  > 章节标题的中文大写数字转半角数字，并尝试以 `0` 补齐位数差异，例：`第八章 → 第08章`

- 02.标题 → 数字转大写

  > 章节标题的半角数字转中文大写数字，例：`第100章 → 第一百章`

### 修正

> 文本常规错误的修正功能

- 01.★修正 → 括号标题

  > 转换括号类型的章节标题为标准样式，例：`（二十） → 第20章`

- 02.★修正 → 小标题括号

  > 转换章节标题最后的小标题为标准括号样式，例：`第11章：大江东去1 → 第11章：大江东去（01）`

- 03.√修正 → 错误标题
  - 【全局状态】尝试清除重复的标题
  - 【全局状态】尝试修正有序列表的文本
  - 【选中状态】尝试修正错误转换的标题

- 11.√修正 → 双引号缺失
  - 选中字符前后补齐双引号

- 12.√修正 → 单引号缺失
  - 【选中字符前后无引号】所选文本中的双引号全转换成单引号
  - 【选中字符前后是引号】所选文本前后全转换成单引号，中间不处理

- 13.★修正 → 引号嵌套
  - 尝试修正双引号中嵌套的双引号为单引号
  - 尝试修正单引号中嵌套的单引号为双引号

- 14.★修正 → 标点缺失
  - 尝试修正段落最后无标点的情况
  - 尝试修正引用中无结尾标点的情况

- 21.★清除 → 广告
- 22.★清除 → 多余分隔符
  - 尝试修正多个分隔符相连的
  - 尝试修正分隔符和章节标题相连的

- 31.★修正 → 拼音缺字
  - 基于屏蔽词组，转换全拼音、拼音首字母组合，进行修正
  - 修正全拼音词组
  - 修正缺字和星号
  - 修正拼音首字母
  - 修正错别字

- 32.★修正 → 词组勘误
  - 修正勘误表词组
  - 修正错别字词组

- 33.★修正 → 词组混淆
  - 基于屏蔽词组，修正清除包含的混淆码

- 34.★修正 → 省市地名
  - 基于省市地名，转换全拼音、拼音首字母组合，进行修正
  - 基于省市地名，修正清除包含的混淆码

- 35.★修正 → 量词小写
  - 不规则的量词小写转换大写

- 41.★修正 → 超长段落分隔

  > 智能分隔：按设定长度进行分隔，句号、引号、省略号进行换行处理，分隔段落不够字数的合并

  - 【全局状态】对超长的段落进行智能分隔，避免段落过长
  - 【选中状态】对选中字符进行智能分隔，并处理引号分行的情况

- 42.★修正 → 强制段落分隔

  > 强制分隔：按规则进行分隔，以句号、引号、省略号后进行换行处理

  - 【全局状态】对超长的段落进行强制分隔，避免段落过长
  - 【选中状态】对选中字符进行强制分隔，并处理引号分行的情况
